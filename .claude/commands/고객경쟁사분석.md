**시작 전**: Bash로 `date +%Y-W%V`를 실행해 현재 주차를 확인해.

CLAUDE.md의 Twelve Senses 프로젝트 컨텍스트를 읽고, WebSearch로 경쟁 제품 동향·고객 리뷰·시장 플레이어를 조사한 뒤, 아래 3가지 방법론으로 **고객·경쟁사 분석 보고서**를 작성해줘.

---

## 방법론 1: Jobs-to-be-Done + Empathy Map — 고객 심층 이해

**JTBD 인터뷰 프레임워크** (WebSearch로 리뷰·SNS 발언 수집):
- **언제** 향기 제품을 "고용"하는가? (상황·맥락)
- **진짜 목적**은? (기능적 / 감성적 / 사회적 Job)
- **현재 대안**은? (캔들, 기존 디퓨저, 아로마 오일 등)
- **전환 동기**는? (기존 대안의 불만족 포인트)
- **성공 정의**는? ("완벽히 해결됐다"는 언제인가?)

**Empathy Map** (4사분면):
| Says (말하는 것) | Thinks (속으로 생각하는 것) |
|----------------|--------------------------|
| **Does (행동)** | **Feels (감정)** |

## 방법론 2: Porter's Five Forces — 산업 구조 분석
| Forces | 강도(1~5) | 근거 | Twelve Senses 시사점 |
|--------|----------|------|---------------------|
| **기존 경쟁자 간 경쟁** | | Dyson Aura, 샤오미, 국내 브랜드 | |
| **신규 진입자 위협** | | 스마트홈 빅테크 진입 가능성 | |
| **공급자 교섭력** | | ESP32, 에센셜 오일 공급업체 | |
| **구매자 교섭력** | | B2C 개인 vs B2B 대형 고객 | |
| **대체재 위협** | | 캔들, 기존 디퓨저, 스마트 조명 | |

→ 산업 매력도 종합 평가 + 진입 장벽 강화 전략
