## 방법론 1: TAM / SAM / SOM — 두 가지 방식으로 교차 검증

### Top-Down (시장 리포트 기반)
| 구분 | 정의 | 수치 | 출처 |
|------|------|------|------|
| **TAM** | 글로벌 홈프래그런스 전체 시장 | $76B(2023) → $110B+(2032), CAGR 4.4~5.65% | 업데이트 필요 |
| **SAM** | 한국 스마트 디퓨저 + 프리미엄 홈프래그런스 | $340M(2024) → $450M+(2035) | 업데이트 필요 |
| **SOM** | Twelve Senses 현실적 점유 가능 시장 | 계산 | |

### Bottom-Up (고객 단위 기반)
- 타겟 고객 수(명) × 평균 구매 가격(원) × 연간 구독 횟수 = SOM
- 시나리오별 계산: 보수적 / 기본 / 낙관적
