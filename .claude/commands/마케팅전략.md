**시작 전**: Bash로 `date +%Y-W%V`를 실행해 현재 주차를 확인해.

CLAUDE.md의 Twelve Senses 프로젝트 컨텍스트를 읽고, `reports/$(date +%Y-W%V)/브랜딩전략.md`와 `고객경쟁사분석.md`가 있으면 읽어 포지셔닝·타겟 인사이트를 반영해. WebSearch로 MZ세대 소비 트렌드·크라우드 펀딩·버티컬 커머스 마케팅 사례를 조사한 뒤, 아래 3가지 방법론으로 **마케팅전략 보고서**를 작성해줘.

## 방법론 1: STP — 세분화·타겟팅·포지셔닝
**Segmentation**: 아래 기준으로 고객 세그먼트 분류
| 세그먼트 | 인구통계 | 심리통계 | 행동 특성 | 시장 규모 |
|---------|---------|---------|---------|---------|
| B2C MZ 얼리어답터 | | 셀프 디깅·레이어링 | | |
| B2C 스몰 럭셔리 | | 가심비 소비 | | |
| B2B 호텔·백화점 | | | 무인 관리 효율 | |

**Targeting**: 이번 단계에서 집중할 세그먼트 + 이유

**Positioning**: 포지셔닝 스테이트먼트
> "[타겟]을 위한 Twelve Senses는 [경쟁 대안]과 달리 [핵심 차별점]을 제공하는 [카테고리]이다."

## 방법론 3: Growth Loop — 자기 강화 성장 설계
Twelve Senses의 바이럴 루프 설계:
```
신규 사용자 가입
      ↓
향기 레시피 생성·저장
      ↓
SNS/친구 공유 (레시피 공유 기능)
      ↓
신규 유입 → 루프 반복
```

- 이번 주 강화할 수 있는 액션?
