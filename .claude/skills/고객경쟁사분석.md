# 고객·경쟁사 분석 스킬

JTBD + Empathy Map으로 고객을 깊이 이해하고, Porter's Five Forces + 직접 비교로 경쟁 구도를 파악하는 스킬.

## 참조 방법론
- [JTBD.md](references/JTBD.md) — 고객이 진짜 원하는 것 (Jobs-to-be-Done)
- [Empathy-Map.md](references/Empathy-Map.md) — 고객 Says/Thinks/Does/Feels
- [Porter-Five-Forces.md](references/Porter-Five-Forces.md) — 산업 경쟁 구조

## 이 스킬을 사용하는 커맨드
- `/고객경쟁사분석`

## 실행 프로세스

### 1단계: JTBD 분석
[JTBD.md](references/JTBD.md)의 5가지 핵심 질문으로 고객 이해:
- WebSearch로 경쟁 제품 리뷰, SNS 댓글, 커뮤니티 반응 수집
- 기능적/감성적/사회적 Job 분류
- 4 Forces(Push/Pull/Anxiety/Inertia) 매핑

### 2단계: Empathy Map
[Empathy-Map.md](references/Empathy-Map.md) 기반 4사분면 완성:
- Says: 리뷰·인터뷰 발언
- Thinks: 말하지 않는 속내 (리뷰 행간 읽기)
- Does: 실제 구매·사용 행동 패턴
- Feels: 감정적 동기·두려움

### 3단계: Porter's Five Forces
[Porter-Five-Forces.md](references/Porter-Five-Forces.md)로 산업 매력도 평가:
- 경쟁사 3~5개 WebSearch로 실제 데이터 조사
- 5개 세력 강도 점수화 (1~5)
- Twelve Senses 전략적 시사점

### 4단계: 경쟁사 직접 비교
| 항목 | Twelve Senses | 경쟁사A | 경쟁사B | 경쟁사C |
|------|-------------|-------|-------|-------|
| 가격 | 미정 | | | |
| 핵심 기능 | PWM+AI+RFID | | | |
| 고객 평점 | - | | | |

### 5단계: 차별화 포인트 정리
JTBD + Porter 분석을 종합해 Twelve Senses만의 차별화 포인트 3가지 도출.

## 품질 기준
- JTBD 분류 3가지 (기능/감성/사회) 모두 커버
- Empathy Map 4사분면 실제 데이터로 채우기 (추측 최소화)
- 경쟁사 최소 3개 이상 실제 조사
- Porter 5요소 모두 강도 점수 + 근거 포함
