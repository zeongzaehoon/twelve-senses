# Golden Circle (Simon Sinek) 방법론

## 개요
Simon Sinek의 "Start With Why" 이론. 위대한 브랜드와 리더는 "무엇(What)"이 아니라 "왜(Why)"부터 소통한다.

## 구조

```
         [WHY]          ← 존재 이유, 신념, 목적
        [HOW]           ← 독특한 실현 방식
       [WHAT]           ← 제품·서비스
```

### 안에서 밖으로 소통 (Inside-Out)
- 일반 브랜드: "우리는 좋은 디퓨저를 만듭니다. AI 기능이 있고, 멋지게 디자인됐어요. 사실 싶지 않으세요?" (What → How → Why)
- 훌륭한 브랜드: "우리는 사람들이 자신만의 감각 세계를 소유할 수 있다고 믿습니다. 그 믿음을 실현하기 위해 AI로 향기를 개인화합니다. 우리가 만든 건 AI 스마트 디퓨저입니다." (Why → How → What)

## 3 레이어 정의

### WHY — 존재 이유
- 돈 버는 것 이상의 목적
- 조직의 근본 신념
- 모든 결정의 기준
- **테스트**: 이 답이 고객에게 영감을 주는가?

### HOW — 실현 방식
- Why를 구현하는 고유한 방법
- 경쟁사가 쉽게 모방 못하는 차별적 접근
- **테스트**: 이것이 실제 우리의 작동 방식과 일치하는가?

### WHAT — 제품/서비스
- 가장 구체적이고 명확한 레이어
- Why와 How의 결과물

## Twelve Senses Golden Circle

| 레이어 | 내용 | 검토 포인트 |
|--------|------|-----------|
| **WHY** | "모든 사람은 자신만의 감각 언어가 있다. 우리는 그 언어를 향기로 번역한다." | 얼마나 공감되는가? |
| **HOW** | AI 학습 + PWM 정밀 조향 + RFID 생태계로 나만의 향 레시피를 실현 | 기술적 차별성 명확한가? |
| **WHAT** | AI 스마트 디퓨저 + 에센셜 오일 구독 + 향기 데이터 플랫폼 | Why와 연결되는가? |

## 마케팅 메시지 재작성
Golden Circle 기반으로 모든 카피라이팅을 Why 중심으로 재작성:

```
기존: "4종 오일 AI 조향 스마트 디퓨저 — 수천 가지 향기 레시피"
개선: "당신만의 향기 취향을 발견하세요 — Twelve Senses가 AI로 만드는 나만의 향"
```

```
### Golden Circle 정의
**WHY**: [한 문장 — 존재 이유]
**HOW**: [2~3문장 — 실현 방식]
**WHAT**: [1문장 — 제품/서비스]

**마케팅 메시지 재작성 제안**:
- 크라우드 펀딩 헤드카피: [Why 중심]
- SNS 바이오: [Why 중심]
- 제품 소개 첫 문장: [Why → How → What 순서]
```
