# Blue Ocean Strategy 방법론

## 개요
W. Chan Kim & Renée Mauborgne(2005)가 제시한 가치 혁신 전략.
기존 경쟁(Red Ocean)을 피해 새로운 시장 공간(Blue Ocean)을 창출.

## 핵심 개념

### Red Ocean vs Blue Ocean
| Red Ocean | Blue Ocean |
|----------|-----------|
| 기존 산업 내 경쟁 | 미개척 시장 창출 |
| 기존 수요 쟁탈 | 새로운 수요 창출 |
| 가치-비용 트레이드오프 | 가치-비용 동시 달성 |
| 차별화 OR 저비용 | 차별화 AND 저비용 |

## 4가지 액션 프레임 (ERRC Grid)

| 액션 | 질문 | 목적 |
|------|------|------|
| **Eliminate (제거)** | 업계가 당연시하는 요소 중 없애도 되는 것? | 비용 절감 |
| **Reduce (감소)** | 업계 표준 이하로 낮춰도 되는 것? | 비용 절감 |
| **Raise (증가)** | 업계 표준 이상으로 높여야 하는 것? | 가치 향상 |
| **Create (창조)** | 업계에 없던 새로운 가치 요소? | 가치 혁신 |

## Twelve Senses ERRC 적용

| 액션 | 적용 |
|------|------|
| **제거** | 수동 향기 조절 (설정 후 방치), 리필 주기 예측 불가 |
| **감소** | 초기 향 종류 선택 복잡성, 디퓨저 메인터넌스 |
| **증가** | 향기 정밀도 (PWM 드롭 단위), 개인화 깊이, 데이터 기반 추천 |
| **창조** | AI 실시간 레시피 생성, RFID 오일 자동 인식, 날씨·감정 기반 조향 |

## 전략 캔버스 (Strategy Canvas)
업계 경쟁 요소를 X축, 제공 수준을 Y축으로 그려 경쟁사 대비 차별점 시각화:
```
제공 수준(高) ┃          ___Twelve Senses___
              ┃       __/                   \__
              ┃   ___/  경쟁사들  __________   \
              ┃__/    ____________             \___
제공 수준(低) ┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
              가격  디자인  향종류  AI  RFID  구독
```

## Pioneer-Migrator-Settler Map
- **Pioneer**: Blue Ocean 창출 (Twelve Senses의 AI 조향)
- **Migrator**: 개선된 가치 (디자인, UX)
- **Settler**: 모방 (기존 디퓨저 기능)
