# Brand Pyramid 방법론

## 개요
브랜드 정체성을 5개 레이어로 구조화하는 프레임워크.
아래에서 위로 올라갈수록 추상적이고, 아래 레이어가 탄탄해야 위 레이어가 설득력을 가짐.

## 5단계 구조

```
        ╔══════════════════╗
        ║  Brand Essence   ║  ← 핵심 정체성 1단어/구문 (모든 것의 요약)
        ╠══════════════════╣
        ║  Brand Vision    ║  ← 우리가 만들고 싶은 미래 세상
        ╠══════════════════╣
        ║  Brand Values    ║  ← 의사결정의 기준이 되는 핵심 가치
        ╠══════════════════╣
        ║  Brand Promise   ║  ← 고객에게 하는 일관된 약속
        ╠══════════════════╣
        ║ Features/Benefits ║ ← 제품 기능이 약속을 어떻게 지키는가
        ╚══════════════════╝
```

## 각 레이어 정의

### Level 1: Features & Benefits (기능과 혜택)
- 제품의 구체적 기능 (HOW)
- 각 기능이 주는 실용적 혜택
- **테스트**: 경쟁사도 이걸 말할 수 있는가? (있다면 차별화 필요)

### Level 2: Brand Promise (브랜드 약속)
- 고객이 항상 기대할 수 있는 경험
- 모든 접점에서 일관되게 전달되어야 함
- **테스트**: 이 약속을 지킬 수 있는가?

### Level 3: Brand Values (브랜드 가치)
- 비즈니스 결정의 필터 역할
- 팀이 공유하는 믿음과 원칙
- **테스트**: 이 가치에 어긋나는 결정을 거부할 수 있는가?

### Level 4: Brand Vision (브랜드 비전)
- 10년 후 세상에 기여하고 싶은 변화
- 존재 이유와 연결된 장기 방향성
- **테스트**: 팀이 이 비전에 동기부여되는가?

### Level 5: Brand Essence (브랜드 에센스)
- 브랜드 전체를 관통하는 1단어 또는 짧은 구문
- 내부 나침반 역할 (외부 슬로건과 달리 내부 언어)
- **테스트**: 이 단어를 들었을 때 우리 브랜드가 연상되는가?

## Twelve Senses Brand Pyramid 초안

| 레이어 | 내용 |
|--------|------|
| **Essence** | "감각의 자유" |
| **Vision** | 모든 공간이 그 사람만의 감각 세계가 되는 세상 |
| **Values** | 초개인화, 탐구, 정밀함, 지속가능한 감각 |
| **Promise** | "당신의 향기 취향을 AI가 기억하고, 매일 완벽하게 재현한다" |
| **Features** | PWM 정밀 조향, AI 레시피 생성, RFID 자동 인식, 날씨 연동 |
