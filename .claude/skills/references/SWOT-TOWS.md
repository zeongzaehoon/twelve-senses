# SWOT + TOWS Matrix 방법론

## 개요
**SWOT**: 내부(강점·약점) + 외부(기회·위협) 요인을 체계적으로 파악하는 분석 도구
**TOWS**: SWOT 결과를 4가지 전략 방향으로 전환하는 실행 도구 (Heinz Weihrich, 1982)

## SWOT vs TOWS 차이

| 구분 | SWOT | TOWS |
|------|------|------|
| 목적 | 현황 분석 | 전략 도출 |
| 방향 | 내부 → 외부 | 외부 → 내부 |
| 결과물 | 요인 목록 | 실행 전략 |

## Weighted SWOT (가중치 분석)

각 요소에 점수를 부여해 우선순위화:
- **영향도 (1~5)**: Twelve Senses에 미치는 영향의 크기
- **통제가능성/발생가능성 (1~5)**: 내부 요인은 통제가능성, 외부는 발생가능성
- **우선순위 점수** = 영향도 × 통제/발생가능성

```
| 요인 | 내용 | 영향도 | 통제/발생 | 점수 |
|------|------|--------|-----------|------|
|  S1  |  ... |   4    |     5     |  20  |
```

## TOWS Matrix

|           | **Opportunities (O)** | **Threats (T)** |
|-----------|----------------------|-----------------|
| **Strengths (S)** | **S-O 전략** (Maxi-Maxi) 강점으로 기회 극대화 | **S-T 전략** (Maxi-Mini) 강점으로 위협 방어 |
| **Weaknesses (W)** | **W-O 전략** (Mini-Maxi) 약점 극복해 기회 포착 | **W-T 전략** (Mini-Mini) 약점·위협 최소화, 철수 고려 |

### 전략 도출 방법
- 각 셀: 강점/약점 리스트 + 기회/위협 리스트를 교차해 "이것을 활용/극복해 저것에 대응"하는 구체적 전략 2~3개
- 우선순위: 점수 높은 요인 조합을 먼저 처리
- 실현 가능성·예상 임팩트·리소스 요구량으로 최종 랭킹

## 동적 SWOT (Dynamic SWOT)
- **지난 주 대비 변화**: 업데이트된 요인 하이라이트
- **3개월 예측**: 현재 트렌드 연장시 SWOT 어떻게 변화?
- **조기 경보 신호 (EWS)**: "이 지표가 바뀌면 위협으로 전환" 정의

## Twelve Senses 상시 SWOT 요소

**Strengths**: PWM 정밀조향, RFID Lock-in 생태계, AI 초개인화, ESP32 저비용  
**Weaknesses**: 초기 자본 부족, 브랜드 인지도 없음, 하드웨어 제조 경험 부족  
**Opportunities**: MZ세대 스몰 럭셔리 트렌드, 스마트홈 확산, B2B 무인 관리 수요  
**Threats**: Dyson 등 대기업 진입, 에센셜 오일 모방 제품, 규제 강화  

## 출력 형식
```
## Weighted SWOT 표
[S/W/O/T 각 3~5개, 점수 포함]

## TOWS Matrix
| S-O | S-T |
| W-O | W-T |

## 이번 주 최우선 전략 3가지
1. [TOWS 셀] — [전략 내용] — [실행 주체/기한]
```
