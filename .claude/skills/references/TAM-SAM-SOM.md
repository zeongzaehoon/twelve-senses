# TAM / SAM / SOM 시장규모 산정 방법론

## 개요
스타트업이 투자자에게 시장 기회를 설득하고, 내부적으로 목표 시장을 정의하기 위한 필수 프레임워크.

## 정의

| 지표 | 이름 | 정의 |
|------|------|------|
| **TAM** | Total Addressable Market | 제품이 100% 점유 시 이론적 최대 시장 규모 |
| **SAM** | Serviceable Available Market | TAM 중 실제로 타겟 가능한 세그먼트 |
| **SOM** | Serviceable Obtainable Market | SAM 중 현실적으로 확보 가능한 점유율 |

## 두 가지 산정 방식

### Top-Down (하향식)
시장 리포트 → 필터링 → SOM 도달
```
글로벌 홈프래그런스 시장 (TAM)
  → 한국 + 프리미엄 세그먼트 (SAM)
    → Twelve Senses 현실적 목표 (SOM)
```
- **장점**: 빠름, 거시적 맥락 제공
- **단점**: 가정이 많아 투자자 설득력 낮음
- **데이터 소스**: Grand View Research, Statista, IBISWorld

### Bottom-Up (상향식)
고객 단위 계산 → 합산 → TAM 역산
```
SOM = 타겟 고객 수 × 평균 판매가 × 구매 빈도
SAM = 접근 가능한 전체 고객 × 위 계산
TAM = 시장 전체 적용
```
- **장점**: 투자자 신뢰도 높음, 운영 인사이트 제공
- **단점**: 고객 데이터 필요, 시간 소요
- **Best Practice**: 두 방식 교차 검증으로 신뢰도 확보

## Twelve Senses 계산 프레임

### Top-Down
| 구분 | 수치 | 근거 |
|------|------|------|
| TAM | 글로벌 홈프래그런스 $76B(2023) → $110B+(2032) | CAGR 4.4~5.65% |
| SAM | 한국 홈프래그런스 $340M(2024) → $450M+(2035) | 프리미엄 AI·스마트 세그먼트 |
| SOM | 초기 크라우드 펀딩 + 버티컬 플랫폼 목표 | 계산 필요 |

### Bottom-Up (예시)
```
B2C SOM = 
  와디즈/텀블벅 얼리어답터 500명 × @299,000원 하드웨어
  + 월 구독 @29,900원 × 12개월 × 유지율 70%
  = 1억 5천만원 (Year 1)

B2B SOM = 
  호텔/백화점 파일럿 5개소 × @2,000,000원 설치
  + 월 유지보수 @200,000원 × 12개월
  = 2,200만원 (Year 1)
```
