# Brand Archetypes 방법론

## 개요
Carl Jung의 집단 무의식 이론을 마케팅에 적용한 프레임워크 (Carol Pearson 발전).
12가지 보편적 성격 유형 중 브랜드에 가장 잘 맞는 아케타입을 선정해 일관된 브랜드 성격 구축.

## 12가지 아케타입

| 아케타입 | 핵심 동기 | 대표 브랜드 | 키워드 |
|---------|---------|-----------|--------|
| **Creator (창조자)** | 자기표현, 창의성 | LEGO, Apple | 혁신, 상상력, 독창성 |

## Twelve Senses 아케타입 분석

### 유력 후보

**Creator (창조자)** ← 1순위 추천
- 이유: 사용자가 자신만의 향기를 "창조"하는 경험이 핵심
- 키워드: 자기표현, 독창적, 예술적, 장인정신
- 위험: 창조의 복잡성이 접근 장벽이 될 수 있음

**Magician (마법사)** ← 2순위
- 이유: AI가 날씨·감정을 읽어 "마법처럼" 완벽한 향을 만들어냄
- 키워드: 변환, 특별한 경험, 비범함
- 위험: 기술적 신뢰성이 뒷받침되어야 함

**Explorer (탐험가)** ← 보조
- 이유: 셀프 디깅, 취향 탐구 MZ세대 가치와 일치
- 키워드: 발견, 자기탐구, 자유로운 취향 실험

### 선정 기준
1. 타겟 고객의 자아 이미지와 일치하는가?
2. 경쟁 브랜드와 차별화되는가?
3. 모든 브랜드 접점에서 일관되게 표현 가능한가?

## 아케타입 적용 — 브랜드 표현 가이드

### Creator 선정 시
| 요소 | 방향 |
|------|------|
| **톤 오브 보이스** | 영감을 주되 지시하지 않음 / 예술적·감성적 언어 / 가능성을 열어두는 표현 |
| **비주얼** | 창의적 레이어링 / 오일이 섞이는 순간 / 개인의 공간 |
| **콘텐츠** | 사용자 레시피 스토리 / 조향 과정 / 향기로 표현하는 감정 |
| **인터랙션** | 사용자가 직접 만드는 UX / 자유로운 탐색 허용 |

## 출력 형식
```
### Brand Archetype 선정
**선정 아케타입**: [이름]
**선정 이유**: [Twelve Senses 맥락에서 3가지 근거]
**2차 아케타입 (보조)**: [이름]

**표현 가이드**:
- 톤 오브 보이스: [3~5가지 형용사]
- 금지 언어: [사용하면 안 되는 표현]
- 비주얼 방향: [색상, 이미지 톤]
- 콘텐츠 방향: [어떤 이야기를 해야 하는가]
```
