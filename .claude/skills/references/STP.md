# STP 마케팅 방법론

## 개요
Segmentation(세분화) → Targeting(타겟팅) → Positioning(포지셔닝)으로 이어지는 마케팅 전략의 핵심 프레임워크.
"맞는 사람에게, 맞는 방식으로, 맞는 시간에" 마케팅하기 위한 구조.

## S — Segmentation (시장 세분화)

### 세분화 기준 4가지

| 기준 | 변수 | Twelve Senses 예시 |
|------|------|-------------------|
| **인구통계** | 연령, 성별, 소득, 직업 | 25~38세, 여성 55%, 월 소득 300만원+ |
| **심리통계** | 라이프스타일, 가치관, 성격 | 셀프 디깅, 스몰 럭셔리, 얼리어답터 |
| **행동** | 구매 동기, 사용 빈도, 충성도 | 향수·향기 레이어링 경험자, 크펀 참여자 |
| **지리** | 지역, 도시 규모 | 서울·수도권 도심 거주 |

### 주요 세그먼트 매트릭스

| 세그먼트 | 특성 | 규모 | 접근성 | 수익성 |
|---------|------|------|--------|--------|
| MZ 얼리어답터 | 셀프 디깅, 테크 친화 | | 크펀/SNS | 높음 |
| 스몰 럭셔리 소비자 | 가심비, 29cm 고객 | | 버티컬 플랫폼 | 중간-높음 |
| B2B 호텔·백화점 | 무인 관리, 공간 브랜딩 | | 직접 영업 | 높음 (대량) |

## T — Targeting (타겟 선정)

### 타겟 선정 기준 (DAMS)
- **D**istinctiveness: 각 세그먼트가 충분히 구별되는가?
- **A**ccessibility: 접근 가능한 채널이 있는가?
- **M**easurability: 규모와 잠재력을 측정할 수 있는가?
- **S**ubstantiality: 수익을 낼 만큼 충분히 큰가?

### Twelve Senses 타겟 전략
- **1차 타겟 (론칭)**: MZ 얼리어답터 → 크라우드 펀딩으로 초기 팬층 확보
- **2차 타겟 (성장)**: 스몰 럭셔리 소비자 → 29cm, EQL 입점
- **3차 타겟 (확장)**: B2B 공간 → 호텔, 백화점 직접 영업

## P — Positioning (포지셔닝)

### Positioning Statement
```
[타겟 고객]을 위한 Twelve Senses는,
[경쟁 대안]과 달리,
[핵심 차별점]을 제공하는
[카테고리]이다.
```

**초안**:
> "자신만의 향기 취향을 탐구하는 MZ세대를 위한 Twelve Senses는,
> 정해진 향을 방향하는 기존 디퓨저와 달리,
> AI가 날씨·감정·취향을 학습해 실시간으로 나만의 레시피를 조향하는
> 세계 최초 AI 스마트 조향 플랫폼이다."

### Perceptual Map (인식 지도)
```
        [하이엔드/프리미엄]
              ↑
[수동·기존] ←──────────── ────→ [AI·스마트]
              ↓
        [로우엔드/일반]
```

## 출력 형식
```
### STP 분석
**타겟 세그먼트**: [이번 단계 집중 세그먼트]
**선정 이유**: [DAMS 기준]
**포지셔닝 스테이트먼트**: [완성된 문장]
**차별화 메시지 3가지**: [채널별 핵심 메시지]
```
