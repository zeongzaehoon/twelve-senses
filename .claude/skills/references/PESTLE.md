# PESTLE 분석 방법론

## 개요
거시환경 6가지 요소를 체계적으로 분석해 기회·위협을 식별하는 프레임워크.
원래 PEST(Political, Economic, Social, Technological)에서 Legal + Environmental을 추가한 확장판.

## 6가지 요소

| 요소 | 핵심 질문 | 분석 방향 |
|------|----------|----------|
| **Political** | 정책·규제·정치적 안정성? | 정부 지원, 규제 변화, 인허가 |
| **Economic** | 거시경제 지표, 소비자 지출? | 금리, 환율, 실업률, GDP 성장 |
| **Social** | 인구통계, 문화·가치관 변화? | 트렌드, 라이프스타일, 세대 특성 |
| **Technological** | 기술 혁신, 디지털 전환? | 신기술 도입, R&D, 자동화 |
| **Legal** | 법률·규제·컴플라이언스? | 노동법, 소비자보호, 지식재산권 |
| **Environmental** | 환경 규제, 지속가능성? | 탄소발자국, 원자재 소싱, 재활용 |

## 적용 절차

1. **스코프 정의** — 분석 범위 (지역: 한국/글로벌, 산업: 홈프래그런스/IoT)
2. **데이터 수집** — 각 요소별 WebSearch로 최신 뉴스·통계 수집
3. **영향도 평가** — 각 요소가 Twelve Senses에 미치는 영향
   - 긍정(O) / 부정(T) / 중립
   - 영향도: 높음·중간·낮음
   - 시급도: 즉시·단기·장기
4. **상호연결 매핑** — 요소 간 인과관계 파악 (시스템적 사고)
5. **Scenario Planning 연결** → [시나리오플래닝.md](시나리오플래닝.md) 참조

## Twelve Senses 적용 예시

| 요소 | 주요 관찰 항목 |
|------|--------------|
| Political | IoT 인증, 스타트업 지원 정책, 크라우드 펀딩 규제 |
| Economic | MZ세대 소비 여력, 금리 변화, 스몰 럭셔리 소비 트렌드 |
| Social | 셀프 디깅, 레이어링, 웰니스, 향기 소비 문화 확산 |
| Technological | ESP32 에코시스템, 생성형 AI 비용, 스마트홈 플랫폼 |
| Legal | 개인정보보호법(PIPA), 화학물질 안전 규정, 제품 인증 |
| Environmental | 에센셜 오일 지속가능 소싱, 포장재 재활용 |

## 정량화 기법 (Quantified PESTLE)
각 요소에 **영향도(1~5) × 발생가능성(1~5) = 우선순위 점수** 부여.
점수 높은 상위 3개 요소를 Scenario Planning의 축으로 사용.

## 출력 형식
```
### [요소명]
- **이번 주 변화**: [최신 뉴스/데이터]
- **Twelve Senses 영향**: [긍정/부정/중립] — [구체적 설명]
- **시급도**: [높음/중간/낮음]
- **SWOT 분류**: O (기회) 또는 T (위협)
```
