# OKR (Objectives & Key Results) 방법론

## 개요
Andy Grove(Intel)가 개발하고 John Doerr가 구글에 도입해 대중화한 목표 설정 프레임워크.
"어디로 갈 것인가(O)"와 "어떻게 알 수 있는가(KR)"를 분리해 명확성과 측정 가능성을 확보.

## 구성 요소

### Objective (목표)
- **정성적**, 영감을 주는 방향성 문장
- 답해야 할 질문: "우리는 무엇을 달성하고 싶은가?"
- 조건: 명확, 도전적, 3개월~분기 단위, 팀이 공감

### Key Results (핵심 결과)
- **정량적**, 측정 가능한 성과 지표
- 답해야 할 질문: "달성됐다는 걸 어떻게 알 수 있는가?"
- 조건: 숫자로 표현, 목표당 2~4개, 달성도 0~1.0 척도
- **60~70% 달성이 이상적** (100% = 목표가 너무 쉬움)

## 좋은 OKR vs 나쁜 OKR

| 나쁜 예 | 좋은 예 |
|--------|--------|
| O: 앱을 잘 만든다 | O: 사용자가 첫 향기 레시피를 만드는 순간을 완벽하게 만든다 |
| KR: 앱 개선 | KR: 첫 레시피 생성까지 평균 시간 5분 이하 달성 |
| KR: 고객 만족 | KR: 첫 주 재방문율 60% 이상 달성 |

## OKR 사이클

```
분기 시작 → OKR 설정 → 주간 체크인(달성도 업데이트) → 중간 점검 → 분기 말 회고
```

- **주간 체크인**: KR 달성도(%) + 신뢰도(%) + 블로커 공유
- **분기 회고**: 달성도 점수, 학습, 다음 분기 반영

## Twelve Senses OKR 예시

**분기 OKR (2026 Q2)**
```
O1: 기술 검증을 완료하고 크라우드 펀딩 론칭 준비를 마친다
  KR1: PWM 4채널 동시 제어 정확도 ±5% 이하 달성
  KR2: 앱-ESP32 BLE 연결 안정성 99% 달성 (100회 테스트)
  KR3: 알파 테스터 10명 NPS 50점 이상

O2: 크라우드 펀딩 성공을 위한 브랜드 기반을 구축한다
  KR1: 와디즈 SNS 팔로워 1,000명 확보
  KR2: 제품 소개 영상 유기적 조회수 5,000회
  KR3: 사전 관심 등록 500명
```
