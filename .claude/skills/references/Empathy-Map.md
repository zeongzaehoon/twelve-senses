# Empathy Map (공감 지도) 방법론

## 개요
XPLANE이 개발한 고객 이해 도구. 고객이 "말하는 것"과 "속으로 생각하는 것"의 간극을 파악해 더 깊은 고객 이해 달성. Design Thinking의 Empathize 단계에서 주로 사용.

## 4사분면 구조

```
┌──────────────────┬──────────────────┐
│   THINKS (생각)   │    FEELS (감정)   │
│                  │                  │
│  속으로 생각하는것  │   느끼는 감정·     │
│  걱정, 진짜 의도   │   두려움, 욕망     │
├──────────────────┼──────────────────┤
│    SAYS (말)      │    DOES (행동)    │
│                  │                  │
│  실제로 말하는 것   │   실제로 하는 행동  │
│  인터뷰·리뷰 발언  │   사용 패턴, 루틴  │
└──────────────────┴──────────────────┘
```

### 추가 요소
- **SEES (보는 것)**: 환경, 시장, 주변 사람들의 행동
- **HEARS (듣는 것)**: 친구/가족/미디어로부터 받는 영향
- **PAINS (고통)**: 두려움, 좌절, 장애물
- **GAINS (이득)**: 원하는 것, 필요한 것, 성공의 측정 기준

## 데이터 수집 방법
- 직접 인터뷰 (1차 데이터)
- 제품 리뷰 분석 (경쟁사 포함)
- SNS 댓글·게시물 모니터링
- 고객 지원 로그 분석
- WebSearch로 커뮤니티 반응 수집

## Twelve Senses 고객 Empathy Map

### 페르소나: "향기 취향 탐구 MZ" (25~33세, 여성, 서울 거주)

| 사분면 | 내용 |
|--------|------|
| **Says** | "향수는 좋아하는데 집에서 오래 지속되는 게 없어" / "내 취향인 향을 정확히 설명하기 어려워" |
| **Thinks** | '캔들 켜놓으면 무섭고, 기존 디퓨저는 너무 강하거나 약하고' / '내 공간에 맞는 향이 뭔지 모르겠어' |
| **Does** | 향수 레이어링 시도, 캔들 컬렉팅, 아로마 숍 방문, 향기 관련 SNS 팔로우 |
| **Feels** | 집에 돌아왔을 때 심리적 전환 욕구, 자신만의 취향을 표현하고 싶은 욕구, 과도한 선택지에 피로 |
| **Pains** | 가격 대비 지속성 부족, 향 강도 조절 불가, 오일 혼합이 번거로움 |
| **Gains** | 귀가 시 즉각적 심리 전환, 취향 있는 사람으로 인식, 집 방문객에게 인상 남기기 |
