# Growth Loop 방법론

## 개요
제품의 성장이 선형이 아닌 복리(compounding) 방식으로 일어나도록 설계하는 프레임워크.
출력(Output)이 다시 입력(Input)으로 재투입되는 폐쇄형 순환 시스템.

## AARRR 퍼널과의 차이

| AARRR 퍼널 | Growth Loop |
|-----------|------------|
| 선형 (A→A→R→R→R) | 순환 (완료 → 다시 시작) |
| 각 단계 독립 최적화 | 시스템 전체 관점 |
| 소진성 성장 | 복리 성장 |

## Growth Loop 유형

### 1. 바이럴 루프 (Viral Loop)
```
사용자 가입 → 핵심 경험 → 초대/공유 → 신규 사용자 유입
```

### 2. 콘텐츠 루프 (Content Loop)
```
사용자 콘텐츠 생성 → 검색 엔진 노출 → 새 유입 → 더 많은 콘텐츠
```

### 3. 제품 주도 루프 (Product-Led Loop)
```
무료 사용 → 가치 경험 → 유료 전환 → 수익 → 제품 개선 → 더 나은 가치
```

### 4. 데이터 루프 (Data/AI Loop)
```
사용자 데이터 → AI 학습 → 더 좋은 추천 → 더 많은 사용 → 더 많은 데이터
```

## Twelve Senses Growth Loop 설계

### 핵심 바이럴 루프
```
신규 사용자
    ↓ 앱 설치 + 디퓨저 연결
첫 AI 레시피 경험 (Activation)
    ↓ "이 향 조합 직접 만들었어"
SNS 공유 / 친구 초대 (Referral)
    ↓ 공유된 레시피 클릭
신규 사용자 유입 → 루프 반복
```

### 데이터 루프 (장기 경쟁 우위)
```
사용자 향기 데이터 축적
    ↓
AI 모델 정교화
    ↓
더 정확한 개인화 레시피
    ↓
더 높은 만족도 + 더 많은 사용
    ↓
더 많은 데이터 → 루프 반복
```

## 루프 강도 측정
- **K-factor (바이럴 계수)** = 초대 수 × 초대 전환율
  - K > 1: 폭발적 성장
  - K = 0.3~0.7: 건강한 보조 성장
- **Loop Velocity**: 루프 1회 완성에 걸리는 시간

## 취약한 연결고리 진단
루프에서 가장 약한 부분 = 전환율이 가장 낮은 단계
→ Twelve Senses 초기 예상 취약점: SNS 공유 단계 (레시피 공유 UX 미완성)

```
**가장 약한 연결고리**: [단계 + 근거]
**이번 주 강화 액션**: [구체적 개선 방법]
```
