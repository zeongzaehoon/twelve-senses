# Porter's Five Forces 방법론

## 개요
Michael Porter(1979)가 개발한 산업 구조 분석 프레임워크.
5가지 경쟁 세력의 강도를 분석해 산업 매력도와 수익성을 평가.

## 5가지 경쟁 세력

```
                    [신규 진입자 위협]
                           ↓
[공급자 교섭력] ←→ [기존 경쟁자 간 경쟁] ←→ [구매자 교섭력]
                           ↑
                    [대체재 위협]
```

### 1. 기존 경쟁자 간 경쟁 (Rivalry)
강도 높아지는 조건: 경쟁자 수 多, 성장률 低, 차별화 少, 고정비 高, 철수 장벽 高

### 2. 신규 진입자 위협 (New Entrants)
진입 장벽 요소: 규모의 경제, 브랜드 충성도, 전환 비용, 자본 요구량, 유통 접근성, 규제

### 3. 공급자 교섭력 (Supplier Power)
강도 높아지는 조건: 공급자 수 少, 전환 비용 高, 공급자가 전방통합 가능, 대체 공급원 없음

### 4. 구매자 교섭력 (Buyer Power)
강도 높아지는 조건: 구매자 수 少, 구매량 大, 전환 비용 低, 구매자가 후방통합 가능

### 5. 대체재 위협 (Substitutes)
강도 높아지는 조건: 대체재 성능 향상, 가격 매력도 高, 전환 비용 低

## 평가 방법
각 세력별 **강도 점수 (1~5)**:
- 1: 매우 낮음 (유리)
- 5: 매우 높음 (불리)

**종합 산업 매력도** = 5개 점수 평균이 낮을수록 매력적

## Twelve Senses 적용

| Forces | 주요 관찰 대상 | 초기 평가 |
|--------|-------------|----------|
| 기존 경쟁자 | Dyson Aura, 샤오미, 국내 아로마 브랜드 | 중간 (3) |
| 신규 진입자 | 빅테크(삼성, LG) 스마트홈 확장 | 높음 (4) |
| 공급자 | ESP32 공급 (Espressif), 에센셜 오일 수입상 | 중간 (3) |
| 구매자 | B2C: 분산, B2B: 집중 | B2C 낮음(2), B2B 높음(4) |
| 대체재 | 캔들, 기존 디퓨저, 자연 환기 | 높음 (4) |

**전략적 시사점**:
- RFID Lock-in → 구매자 전환 비용 증가 (구매자 교섭력 낮춤)
- AI 맞춤화 → 차별화로 대체재 위협 방어
- B2B SaaS → 공급자 의존도 분산 필요
