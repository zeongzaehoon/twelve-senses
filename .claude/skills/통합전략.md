# 통합전략 분석 스킬

SWOT/TOWS + OKR + Balanced Scorecard + Blue Ocean을 결합해 Twelve Senses의 실행 전략을 도출하는 스킬.

## 참조 방법론
- [SWOT-TOWS.md](references/SWOT-TOWS.md) — 강점·약점·기회·위협 + 전략 도출
- [OKR.md](references/OKR.md) — 목표와 핵심 결과 설정
- [Balanced-Scorecard.md](references/Balanced-Scorecard.md) — 4관점 전략 균형 점검
- [Blue-Ocean.md](references/Blue-Ocean.md) — 비경쟁 가치 혁신 전략

## 이 스킬을 사용하는 커맨드
- `/SWOT분석` (SWOT/TOWS)
- `/전략기획` (OKR + BSC + Blue Ocean)

## 실행 프로세스

### [SWOT분석] 프로세스
1. **입력 읽기**: `reports/$(date +%Y-W%V)/PEST분석.md`, `시장조사.md`, `고객경쟁사분석.md`
2. **Weighted SWOT**: [SWOT-TOWS.md](references/SWOT-TOWS.md) 기준 S/W/O/T 각 3~5개, 점수 산출
3. **TOWS Matrix**: 4셀 전략 2~3개씩 도출
4. **동적 SWOT**: 지난 주 대비 변화 + 3개월 예측

### [전략기획] 프로세스
1. **OKR 점검**: [OKR.md](references/OKR.md) — 이번 분기 OKR 달성도 업데이트
2. **BSC 점검**: [Balanced-Scorecard.md](references/Balanced-Scorecard.md) — 4관점 KPI 현황
3. **Blue Ocean**: [Blue-Ocean.md](references/Blue-Ocean.md) — ERRC Grid 업데이트
4. **통합 우선순위**: TOWS 전략 + OKR KR + BSC 병목 → 이번 주 최우선 3가지

## 품질 기준
- SWOT: S/W/O/T 각 3개 이상, 점수 기반 우선순위 명시
- TOWS: 4셀 모두 전략 2개 이상
- OKR: 달성도(%) + 신뢰도(%) + 블로커 포함
- BSC: 4관점 모두 현황 상태 (🟢/🟡/🔴)
- Blue Ocean: ERRC 4가지 모두 업데이트
