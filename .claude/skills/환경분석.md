# 환경분석 스킬

거시환경(PESTLE)과 미래 불확실성(시나리오 플래닝)을 결합해 Twelve Senses의 기회와 위협을 식별하는 분석 스킬.

## 참조 방법론
- [PESTLE.md](references/PESTLE.md) — 거시환경 6요소 분석
- [Scenario-Planning.md](references/Scenario-Planning.md) — 불확실성 시나리오 매트릭스

## 이 스킬을 사용하는 커맨드
- `/PEST분석`

## 실행 프로세스

### 1단계: 데이터 수집
`reports/$(date +%Y-W%V)/_리서치.md`를 먼저 읽어 기존 데이터 활용.
없으면 WebSearch 6회로 수집 (각 PESTLE 요소별 1~2회).

### 2단계: PESTLE 분석
[PESTLE.md](references/PESTLE.md)의 분석 프레임 적용:
- 6요소별 최신 변화 파악
- 영향도 × 발생가능성 점수 산출
- O(기회) / T(위협) 분류

### 3단계: 시나리오 플래닝
[Scenario-Planning.md](references/Scenario-Planning.md) 프레임 적용:
- 점수 높은 불확실성 요소 2개 선정
- 2×2 시나리오 매트릭스 작성
- 각 시나리오별 전략 대응 방향 도출

### 4단계: SWOT 피드
도출된 O/T를 SWOT분석에 전달할 형식으로 정리.

## 품질 기준
- PESTLE 6요소 모두 커버 (최소 3요소 이상 이번 주 업데이트 있어야 함)
- 시나리오 4개 모두 전략 방향 포함
- WebSearch 출처 URL 포함
- 점수 기반 우선순위 상위 3개 명시

## 에이전트 활용
복잡한 분석이 필요한 경우 [`경영전략분석가`](.claude/agents/경영전략분석가.md) 에이전트에 위임 가능.
