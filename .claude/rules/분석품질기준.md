# 분석 품질 기준

각 분석 보고서가 갖춰야 할 최소 품질 기준.

## 공통 품질 기준

### 데이터 품질
- [ ] 모든 수치에 출처 URL 포함
- [ ] 추정값은 `[추정]` 표시 + 산출 근거
- [ ] 발행 연도 표기된 최신 데이터 우선
- [ ] 지난 주 대비 변화량 명시 (가능한 경우)

### 분석 깊이
- [ ] 분석표 빈칸 없음 (최소 데이터 또는 추정값으로 채움)
- [ ] 원인 분석 포함 (단순 현상 나열 금지)
- [ ] Twelve Senses 특화 시사점 도출 (일반론 금지)

### 실행 연결
- [ ] 실행 가능한 액션 아이템 (최소 3개)
- [ ] 담당자 또는 다음 단계 명시
- [ ] 우선순위 있는 정렬

## 보고서별 기준

### _리서치
- WebSearch 최소 6회 (6개 영역 각 1회)
- 핵심 발견 영역별 3~5개
- Top 5 인사이트 (전체 요약)

### 시장조사
- TAM/SAM/SOM: Top-Down + Bottom-Up 두 방식 모두
- 시장 신호: 최소 5개
- 크라우드 펀딩 플랫폼 유사 제품 비교

### PEST분석
- PESTLE 6요소 모두 커버
- 각 요소: 영향도 + 시급도 점수
- 시나리오 4개 모두 전략 방향

### 고객경쟁사분석
- JTBD 3가지 Job 유형 모두
- 경쟁사 최소 3개 실제 조사
- Porter 5요소 강도 점수 + 근거

### SWOT분석
- S/W/O/T 각 최소 3개 항목
- Weighted SWOT 점수 포함
- TOWS 4셀 × 전략 2개 이상
- 동적 SWOT: 지난 주 대비 변화

### 전략기획
- OKR: 달성도 + 신뢰도 + 블로커
- BSC: 4관점 모두 현황 상태
- Blue Ocean: ERRC 4가지 업데이트

### 제품기획
- JTBD 기반 피처 정의
- RICE Score 상위 5개
- MoSCoW 분류 완성

### 브랜딩전략
- Golden Circle 3레이어 완성
- Brand Pyramid 5레이어 완성
- Archetype 선정 + 적용 가이드

### 마케팅전략
- STP 포지셔닝 스테이트먼트 완성
- AARRR 5단계 지표값
- Growth Loop K-factor
