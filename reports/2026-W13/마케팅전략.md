### 1-3. Positioning Statement

**B2C 포지셔닝**

> "취향 탐구와 자기 표현을 중시하는 MZ세대에게, Twelve Senses는 **AI 스마트 조향 플랫폼**이다. 왜냐하면 세계 최초로 4채널 PWM 마이크로 드롭 기술과 생성형 AI가 결합되어 수천 가지 개인화 레시피를 실현하기 때문이다. 기존 디퓨저들이 브랜드가 정해놓은 단일 향을 소비하게 하는 것과 달리, Twelve Senses는 사용자가 직접 자신만의 시그니처 향을 창조하고 AI가 쓸수록 더 정교하게 진화하는 유일한 솔루션이다."

**B2B 포지셔닝**

> "공간 브랜딩의 일관성과 운영 효율을 동시에 요구하는 호텔·백화점·리조트 운영자에게, Twelve Senses는 **무인 통합 향기 관리 SaaS**이다. 왜냐하면 RFID 기반 원격 모니터링·자동 발주·레시피 A/B 테스트를 단일 앱으로 제공하기 때문이다. 수동 교체형 디퓨저나 고비용 향기 컨설팅 업체와 달리, 본사에서 전 지점의 향 품질을 실시간으로 관리하면서 향기 효과를 데이터로 증명할 수 있는 유일한 솔루션이다."

## 2. AARRR 퍼널 설계

> **설계 원칙**: B2C 핵심 타겟 (S1+S2) 기준. 각 단계는 상호 연결되어 Growth Loop를 형성.

| 단계 | 핵심 KPI | 목표값 (2026 연말 기준) | 핵심 채널 / 전술 | 콘텐츠 유형 |
|------|---------|-------------------|----------------|-----------|
| **Acquisition** (인지·유입) | 월 유니크 방문자 (랜딩페이지+앱스토어) / SNS 팔로워 / 와디즈 사전 알림 신청 | 월 방문자 20,000명 / 팔로워 10,000명 / 사전 알림 1,000명 | 인스타그램·릴스 (숏폼 향기 시각화) / 29CM 신상입주 / 와디즈 펀딩 / 오늘의집 향테리어 큐레이션 / 인플루언서 협업 (향기·인테리어 계정 팔로워 5만+) | 15~30초 숏폼 릴스 (향 생성 과정), 인플루언서 리뷰, 플랫폼 신상입주 콘텐츠 |
| **Activation** (첫 경험 활성화) | 앱 설치 후 7일 내 첫 레시피 생성 완료율 / 온보딩 완주율 | 온보딩 완주율 70% / 7일 내 레시피 생성 60% | 취향 인터뷰 5~7문항 → AI 첫 레시피 3종 즉시 제안 / "세상에 하나뿐인 레시피가 완성됐습니다" 마법적 온보딩 모먼트 설계 / RFID 첫 스캔 경험 (원산지·성분 표시) | 온보딩 가이드 인터랙션, 앱 내 "첫 번째 레시피 완성" 축하 애니메이션 |
| **Retention** (재방문·재사용) | 월 앱 접속 횟수 / 6개월 리텐션율 / 오일 구독 유지율 | 월 앱 접속 15회+ / 6개월 리텐션 60% / 구독 6개월 유지율 70% | "오늘의 향 추천" 일일 푸시 알림 (날씨 연동) / "나의 향 DNA" 분기별 리포트 / "지난 봄 즐겼던 향, 올봄에도?" 개인화 재구매 알림 / 레시피 아카이브 누적 자산 시각화 | 개인화 푸시 알림, 앱 내 취향 프로파일 인포그래픽, 네이버 블로그 장기 사용 후기 |
| **Revenue** (수익 전환) | 하드웨어 전환율 / 오일 카트리지 구독 전환율 / 객단가 | 와디즈 후원자 → 정식 구매 전환 40% / 구독 전환율 50% / 연 객단가 35만원+ | RFID 오일 정기 구독 (Lock-in 생태계) / 얼리버드 → 정가 전환 프로모션 / 다공성 세라믹 카트리지 정기 교체 유도 / B2B SaaS 월정액 | 오일 잔량 알림 → 재구매 CTA, 구독 절약 금액 시각화, B2B 도입 ROI 계산기 |
| **Referral** (추천·바이럴) | UGC 생성 수 / 해시태그 게시물 수 / NPS (순추천지수) | 월 UGC 100건+ / #나의향DNA 해시태그 누적 1,000건 (2026 연말) / NPS 50+ | "나의 향 DNA" SNS 공유 카드 (Spotify Wrapped 형식) / 레시피 공유 시 친구 초대 인센티브 / 오리지널 멤버 커뮤니티 (와디즈 후원자 전환) | 공유 가능한 취향 카드 (레이더 차트), 레시피 공유 링크, UGC 리그램 |

### AARRR 단계별 병목 포인트 및 대응

| 병목 구간 | 예상 원인 | 대응 전술 |
|---------|---------|---------|
| Acquisition → Activation | 앱 설치 후 하드웨어 미보유자의 체험 제한 | 앱 단독 "향 취향 진단" 기능 제공 (하드웨어 없이도 레시피 추천 체험 가능) |
| Activation → Retention | AI 초기 추천 정확도 낮음 → 이탈 | 온보딩 취향 인터뷰 정밀도 강화 + 첫 2주 적극 피드백 수집 팝업 |
| Retention → Revenue | 오일 소진 후 재구매 동기 부족 | 잔량 20% 시점 개인화 추천 알림 + 구독 전환 시 오일 1개 증정 |
| Revenue → Referral | 공유 동기 약함 | "향 DNA 리포트" 등 소셜화 가능한 자산 생성 주기 단축 (분기 → 월 1회) |

## 3. Growth Loop 설계

### 3-1. B2C 바이럴 루프

```
[신규 유입]
    ↓
앱 온보딩 → AI 취향 인터뷰 → 첫 레시피 생성 (마법적 모먼트)
    ↓
RFID 오일 구독 시작 → 일상 사용 데이터 축적
    ↓
"나의 향 DNA" 시각화 → SNS 공유 (인스타·카카오채널)
    ↓
공유 콘텐츠 → 신규 유입 (팔로워·지인 Acquisition) ←─┐
    ↓                                                  │
AI 레시피 품질 개선 (데이터 누적 효과)                  │
    ↓                                                  │
사용자 만족도 상승 → NPS 상승 → 구전 추천 ─────────────┘
```

**각 단계 구체화 및 병목 포인트**

| 루프 단계 | 핵심 동인 | 병목 포인트 | 강화 방법 |
|---------|---------|----------|---------|
| 신규 유입 → 온보딩 | 플랫폼 큐레이션·인플루언서 | 앱 설치 마찰 | 와디즈·29CM 랜딩 → 원클릭 앱 연동 |
| 온보딩 → 첫 레시피 | AI 즉시 반응 속도 | 취향 인터뷰 이탈 | 5문항 이하로 압축, 진행률 시각화 |
| 사용 → 데이터 축적 | 일일 발향 습관 형성 | 앱 접속 동기 부족 | "오늘의 향 추천" 푸시로 일일 접촉점 확보 |
| 데이터 → SNS 공유 | 공유 가능한 자산(향 DNA) | 공유 콘텐츠 완성도 낮음 | 인스타 최적화 공유 카드 자동 생성, 클릭 1회 공유 |
| 공유 → 신규 유입 | 콘텐츠 바이럴성 | 공유 카드 소비 후 행동 유도 미약 | 공유 링크에 "향 취향 진단 무료 체험" CTA 삽입 |
| 데이터 → AI 개선 | 데이터 볼륨·품질 | 초기 데이터 부족 (Cold Start) | 온보딩 취향 인터뷰 + 커뮤니티 레시피 크라우드소싱으로 Cold Start 극복 |

---

### 3-2. B2B 확장 루프

```
[B2B 파트너 유치]
    ↓
공간 설치 (호텔·백화점·리조트) → 방문객 향기 경험
    ↓
"이 향 어디 거예요?" → 브랜드 노출 + B2C 유입 증가
    ↓
B2C 구매 증가 → 사용 데이터 축적 → AI 레시피 품질 향상
    ↓
Big Data Hub → 향기 인사이트 리포트 (날씨별·계절별·공간별 트렌드)
    ↓
리포트 가치 상승 → 더 많은 B2B 파트너 유치 ←──────────┐
    ↓                                                    │
B2B 케이스 스터디 (체류 시간·재방문율 상승 데이터) ─────────┘
```

**각 단계 구체화 및 병목 포인트**

| 루프 단계 | 핵심 동인 | 병목 포인트 | 강화 방법 |
|---------|---------|----------|---------|
| B2B 파트너 유치 | 영업력·박람회 노출 | 의사결정 사이클 길음 (3~6개월) | 무료 파일럿 패키지 제공 (3개월 평가 후 계약), 박람회 현장 계약 인센티브 |
| 공간 설치 → 방문객 경험 | 하드웨어 신뢰성·향 품질 | 설치 초기 향 조율 시간 | 전담 온보딩 매니저 + 공간별 레시피 설정 가이드 |
| 방문객 경험 → B2C 유입 | 브랜드 노출·QR 연동 | 브랜드 인지 연결 부족 | 공간 내 QR 코드 ("이 향 정보 보기" → 앱 설치 유도), 파트너 브랜드 공동 SNS 콘텐츠 |
| 데이터 → 인사이트 리포트 | 데이터 볼륨 충분성 | 초기 B2B 계약 수 부족 → 데이터 희소 | B2C 데이터와 B2B 데이터 통합 분석으로 초기 리포트 가치 확보 |
| 리포트 → 파트너 유치 | 케이스 스터디 신뢰도 | 정량적 성과 데이터 부재 | 파일럿 계약 시 "향기 효과 측정 KPI" 사전 합의 + 결과 공동 발표 |

---

### 3-3. 두 루프의 상호 강화 구조

```
B2C 루프 ──→ 데이터 축적 ──→ AI 개선 ──→ B2C 만족도 상승
                    ↑↓
              Big Data Hub
                    ↑↓
B2B 루프 ──→ 공간 데이터 ──→ 인사이트 리포트 ──→ B2B 확장
              ↑
         B2C 유입 증가 (B2B 공간 노출 효과)
```

두 루프는 Big Data Hub를 공유하며 서로를 강화한다. B2C 사용자가 늘수록 AI 레시피 품질이 높아져 B2B 공간에서도 더 정교한 향기를 제공하고, B2B 공간의 브랜드 노출이 B2C 유입을 늘리는 플라이휠 구조다.
