## 1. 디자인 컨셉 방향

### 1-1. Design Sprint — 사용자 여정 핵심 모먼트 (Map)

| 모먼트 | 사용자 상황 | 디자인 기회 포인트 |
|--------|-----------|-----------------|
| **제품 발견** | 29CM·와디즈 탐색, SNS 피드 스크롤 | 첫 이미지 0.3초 내 "이게 뭐야?" 반응 유도 — 조형적 독창성이 클릭의 결정자 |
| **구매** | 버티컬 플랫폼 상세 페이지, 가격 검토 | 하드웨어 소재·마감 디테일 시각화 + AI 기능 작동 방식 모션 GIF로 기술력 납득 |
| **언박싱** | 제품 수령, 첫 개봉 경험 | 패키징 자체가 SNS 공유 트리거 — "올리고 싶다"는 감정 설계 |
| **첫 사용** | 앱 설치 → 취향 인터뷰 → 첫 레시피 실행 | 5분 안에 "이건 나를 위한 것"이라는 확신 제공 — 온보딩 UX가 결정적 |
| **일상 사용** | 퇴근 후 귀가, 날씨 변화 감지, 기분 전환 | 홈 화면 한눈에 현재 레시피 + 추천 레시피 + 오일 잔량 파악 → 마찰 0 경험 |

---

### 1-2. 컨셉 스케치 3개 방향 (Sketch)

#### 방향 A — "테크 오브제 (Tech Object)"
- **핵심 감성**: 미니멀·프리미엄·정밀함
- **시각 언어**: 완벽한 정육면체 실루엣, 소재 질감 대비(무광 메탈 + 반투명 아크릴), 단색 절제 팔레트
- **앱 스타일**: 다크 기반 인터페이스, 수치 기반 시각화, 조향 슬라이더의 정밀 제어 느낌
- **레퍼런스**: Bang & Olufsen, Teenage Engineering, Nothing Phone

#### 방향 B — "향기의 실험실 (Lab)"
- **핵심 감성**: 과학적·큐레이션·탐구
- **시각 언어**: 플라스크·튜브 형태 모티프, 라벨 타이포그래피, 반투명 오일 레이어 표현
- **앱 스타일**: 라이트 기반 인터페이스, 향 성분 데이터 카드, 레시피 조합 수식 표현
- **레퍼런스**: 향수 아틀리에, 과학 실험 키트, Notion·Linear 스타일 UI

#### 방향 C — "나만의 향 정원 (Garden)"
- **핵심 감성**: 유기적·개인화·감성
- **시각 언어**: 곡선 실루엣, 식물성 텍스처, 계절·날씨 연동 비주얼
- **앱 스타일**: 웜톤 라이트 인터페이스, 일러스트 기반 온보딩, 날씨 아이콘과 향 연결 시각화
- **레퍼런스**: Lush, Aesop, 자연주의 라이프스타일 앱

### 1-3. 선택된 방향 및 근거 (Decide)

**선택: 방향 A "테크 오브제" — 단, 방향 B의 탐구·큐레이션 감성을 앱 UX에 레이어로 통합**

#### 선택 근거

| 판단 기준 | 방향 A 평가 | 비고 |
|----------|-----------|------|
| **포지셔닝 정합성** | 하이엔드 Tech Object 포지셔닝과 완전 일치 | 고객경쟁사분석 JTBD #3: "홈테리어 오브제 + SNS '이게 뭐야' 반응" |
| **경쟁사 차별화** | 코코도르(전통 스틱), Xiaomi(가성비 플라스틱)와 시각적 차별화 명확 | 유일하게 Bang & Olufsen 급 조형 언어 선점 가능 |
| **29CM·EQL 입점 적합성** | 버티컬 플랫폼 큐레이션 철학(심미성·희소성)과 정합 | SWOT S6: 하이엔드 포지셔닝이 29CM 거래액 11배 상승 데이터와 연결 |
| **MZ 셀프 기프팅** | "이 정도는 나를 위해 써도 돼"라는 자기 합리화 유발 | Empathy Map: 77% 셀프 기프팅 경험 |
| **SNS 공유 유인** | 정육면체 미니멀 오브제는 인스타·핀터레스트 피드에 최적화 | 개봉샷·책상샷·책상 위 오브제샷 콘텐츠 포맷 자연 발생 |

#### 핵심 디자인 원칙 3개

> **원칙 1 — Quiet Confidence (조용한 존재감)**
> 외치지 않아도 보인다. 불필요한 장식을 모두 제거한 뒤 남은 형태와 소재만으로 프리미엄을 전달한다. 로고, 텍스트, 장식 요소는 최소화하고 공간과 비례가 말하게 한다.

> **원칙 2 — Invisible Technology (보이지 않는 기술)**
> 기술은 경험 안에 녹아있어야 한다. PWM 드롭 제어, RFID 인증, AI 추천 — 이 모든 복잡성은 사용자에게 단순하고 직관적인 결과로만 보여야 한다. 앱의 인터랙션은 매끄럽고, 하드웨어의 동작은 소리 없이 정확해야 한다.

> **원칙 3 — Owned Experience (소유된 경험)**
> 이 제품과 앱은 "나만의 것"이어야 한다. 색상, 레시피, 향 DNA, 인터페이스 — 개인화 요소를 시각적으로 드러내 사용자가 자신의 정체성과 제품을 동일시하도록 설계한다.

## 2. 하드웨어 디자인 방향

### 2-1. 형태 언어 (Form Language)

| 항목 | 방향 | 세부 내용 |
|------|------|---------|
| **기본 형태** | 완벽한 정육면체 | 15×15×15cm 규격 자체가 아이덴티티 — 모든 면이 동등한 존재감을 가지는 큐브 |
| **모서리 처리** | 마이크로 챔퍼 (1~2mm 모따기) | 완전한 직각의 산업적 느낌과 프리미엄 마감의 중간점 — 날카롭지 않되 유약하지 않음 |
| **표면 구성** | 95% 솔리드 + 5% 기능 노출 | 배출구·RFID 스캔 영역·전원 인디케이터만 최소 노출, 나머지는 완전 플러시 |
| **발향 방식 표현** | 상단 슬릿(미세 격자) | 공기의 흐름을 시각화하는 미세 슬릿 패턴 — 기능과 조형의 일치 |
| **무게감** | 시각적 무게감 강조 | 하단 가중 베이스(고무 피트 내장), 들었을 때 "묵직함"이 품질로 인식되도록 설계 |
| **크기 비례** | 손바닥보다 약간 큰 오브제 | 책상 위·사이드 테이블 위에서 존재감 있되 공간을 장악하지 않는 비례 |

### 2-2. 소재 팔레트

| 파트 | 1안 (론칭 에디션) | 2안 (프리미엄 에디션) | 선택 기준 |
|------|----------------|---------------------|---------|
| **외장 바디** | 무광 ABS + 소프트터치 코팅 | 알루미늄 다이캐스팅 (샌드블라스트 처리) | 1안: 크라우드펀딩 단가 최적화 / 2안: 플래그십·B2B 납품용 |
| **상단 슬릿 패널** | 반투명 스모크 아크릴 | 헤어라인 메탈 메시 | 1안: 내부 LED 인디케이터 확산 효과 / 2안: 프리미엄 마감 |
| **베이스 플레이트** | 무광 블랙 ABS + 논슬립 고무 피트 4개 | 두께 2mm 알루미늄 베이스 | 안정감·무게감 강조 |
| **카트리지 도어** | 마그네틱 플러시 도어 (무광 소재 동일) | — | 측면 또는 후면, 손톱 없이 개폐 가능한 마그네틱 클로저 |
| **표면 마감** | 무광(Matte) 100% | 무광 90% + 글로시 인디케이터 링 10% | 지문 최소화, 오염 내성 우선 |

### 3-3. Molecules (분자) — 복합 컴포넌트

#### M-01. 향 강도 슬라이더 (Scent Intensity Slider)

```
구성: 라벨(향 이름) + 향 계열 아이콘 + 슬라이더 트랙 + 현재값 배지
상태: 기본 / 활성(드래그 중) / 비활성(오일 없음)
인터랙션: 드래그 시 햅틱 피드백(경계값 진동) + 실시간 숫자 업데이트
색상: 슬라이더 채움 색상이 향 계열 Accent 컬러로 자동 매핑
      (우디 → Warm Amber / 시트러스 → Cool Blue / 플로럴 → Sage Green / 오리엔탈 → Accent Primary)
```

#### M-02. 오일 카드 (Oil Cartridge Card)


### 5-2. 하드웨어 디자인 검증

#### 무엇을 검증할 것인가

| 검증 항목 | 검증 질문 | 성공 기준 |
|----------|---------|---------|
| **첫인상** | 사진 3초 노출 후 "이 제품이 어떤 제품 같나요?" | 70% 이상이 "프리미엄 전자기기" 또는 "인테리어 오브제"로 인식 |
| **가격 적정성** | 이 디자인과 기능에 기꺼이 지불할 가격은? | 평균 응답 15만원 이상 |
| **구매 의향** | 현재 가격(15~20만원 가정)에 구매 의향 있나요? | 30% 이상 "매우 그렇다" |
| **SNS 공유 의향** | 이 제품을 받으면 인스타그램에 올리겠나요? | 50% 이상 "그렇다" |
| **컬러 선호도** | Obsidian Black vs. Ivory Mist 중 선택 | 비율 및 이유 데이터 수집 → SKU 생산 비율 결정에 반영 |

#### 어떻게 검증할 것인가

1. **온라인 컨셉 테스트 (2026 Q2 초)**: 3D 렌더링 이미지 + 30초 영상 → 구글 폼 15문항 서베이 (목표 n=100)
2. **소규모 FGI (2026 Q2 중)**: 코어 타겟 6~8명, 3D 출력 목업(목재 큐브 15cm) 실물 체험 → 반응 관찰 + 인터뷰 (60분)
3. **SNS A/B 테스트 (2026 Q2 말)**: 인스타그램 유료 광고 2종 — 배경이 다른 제품샷 → 클릭률·저장률 비교

### 5-3. 앱 UX 디자인 검증

#### 무엇을 검증할 것인가

| 검증 항목 | 검증 질문 | 성공 기준 |
|----------|---------|---------|
| **온보딩 완주율** | 취향 인터뷰 5~7 질문을 끝까지 완료하는가? | 85% 이상 완주 |
| **첫 레시피 만족도** | AI가 제안한 첫 레시피가 마음에 드나요? | 70% 이상 4/5점 이상 |
| **레시피 빌더 직관성** | 슬라이더로 4채널 조작 시 도움 없이 가능한가? | 90% 이상 설명 없이 조작 성공 |
| **향 DNA 공유 의향** | 이 차트를 SNS에 올리고 싶나요? | 40% 이상 "그렇다" |
| **RFID 인증 메시지 이해** | "정품 인증" 배지를 보고 의미를 이해하는가? | 80% 이상 "신뢰감 높아진다" |

#### 어떻게 검증할 것인가

1. **Figma 프로토타입 사용성 테스트 (2026 Q2)**: 핵심 플로우 3개 (온보딩 → 첫 레시피 / 레시피 빌더 / 오일 교체 알림) → 사용자 5명 Think-Aloud 세션 (각 45분)
2. **앱 베타 클로즈드 테스트 (2026 Q3)**: 하드웨어 목업 + 앱 베타 연동 → 코어 타겟 20명 2주 실사용 → 일일 5분 다이어리 앱으로 경험 기록
3. **온보딩 A/B 테스트**: 5문항 vs. 7문항 취향 인터뷰 → 완주율·첫 레시피 만족도 비교
