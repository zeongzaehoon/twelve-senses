## 2. Empathy Map

> 타겟: MZ세대 B2C 핵심 고객 (25~35세, 취향 소비자)

| **Says (말하는 것)** | **Thinks (속으로 생각하는 것)** |
|---|---|
| "향이 좋은 공간에 있으면 기분이 달라져" | '내 취향을 정확히 표현할 향을 아직 못 찾았어' |
| "요즘은 향도 레이어링 해서 쓰는 게 트렌드잖아" | '근데 어떻게 섞어야 좋은지 모르겠고 시행착오가 무서워' |
| "이 디퓨저 너무 예뻐서 샀는데 향이 맘에 안 들어" | '매번 새 디퓨저 사는 건 낭비인 것 같은데 어쩔 수 없잖아' |
| "스마트홈 연동되는 게 있으면 좋겠다" | 'AI가 내 취향을 학습해준다면 진짜 편할 것 같은데 그게 가능해?' |
| **Does (행동)** | **Feels (감정)** |
| 향수·디퓨저 리뷰 유튜브·릴스 정기 시청 | 향 선택 실패 시 낭비감·아쉬움 |
| 오늘의집·29CM에서 인테리어 오브제 탐색 | 고급스러운 물건을 소유할 때의 설렘 |
| 향기 제품 구매 전 성분·리뷰 꼼꼼히 확인 | AI·자동화에 대한 기대와 미지의 불안 공존 |
| 셀프 선물·나를 위한 소비에 기꺼이 지출 (77% 경험) | "이 정도는 나를 위해 써도 돼"라는 자기 합리화 |

## 3. Porter's Five Forces 분석

### 분석표

| Forces | 강도 (1~5) | 근거 | Twelve Senses 시사점 |
|--------|:---------:|------|---------------------|
| **기존 경쟁자 간 경쟁** | ★★☆☆☆ (2) | 국내 스마트 디퓨저 전문 플레이어 부재 확인. 코코도르·탬버린즈는 IoT 기능 없는 전통 향기 브랜드. Pura·Vitruvi는 국내 미출시 수준. | 선점 기회 명확. 단, 2~3년 내 대형 가전사(삼성·LG) 진입 시 경쟁 급등 가능 |
| **신규 진입자 위협** | ★★★☆☆ (3) | 향기 + IoT + AI 통합 개발 난이도는 높지만, 대형 플레이어(삼성SmartThings, LG ThinQ, Xiaomi) 향기 카테고리 진입 가능성 상존. 가전사 유통망·자본력 보유. | 특허·RFID Lock-in 생태계 조기 구축 필수. 브랜드 팬덤 선점이 핵심 방어선 |
| **공급자 교섭력** | ★★★☆☆ (3) | BLDC 펌프·ESP32·RFID 모듈은 복수 공급사 존재 (교섭력 낮음). 단, 프리미엄 에센셜 오일 원료 공급사는 제한적 (교섭력 중상). | 에센셜 오일 원료 복수 공급처 확보, 장기계약 선행 권장. RFID 태그 내재화 검토 |
| **구매자 교섭력** | ★★★☆☆ (3) | B2C: 가격 민감도 있으나 하이엔드 포지셔닝 시 가격 협상력 낮음. B2B: 호텔·백화점 등 대형 고객은 가격 협상력 높음 + 전환 비용 요구. | B2C는 프리미엄 가격 정당화 (디자인·AI 기능). B2B는 Lock-in 계약 구조 (구독형 SaaS) 설계 |
| **대체재 위협** | ★★★★☆ (4) | 캔들·스틱 디퓨저·향수·아로마 가습기 등 향기 전달 수단 다양. 스틱 디퓨저 단가 8,900원(코코도르) 대비 스마트 디퓨저는 고가. | 기능적 대체재보다 "경험 카테고리" 포지셔닝 필요. "AI 조향 플랫폼"으로 카테고리 자체를 새로 정의 |

### 산업 매력도 종합 평가

- **종합 점수**: 15 / 25 (매력도 중상)
- **핵심 기회**: 국내 스마트 디퓨저 공백 (기존 경쟁 강도 낮음) + 홈프래그런스 1조원 시장 + MZ 셀프 선물 트렌드
- **핵심 위협**: 대체재 위협 높음 (저가 대안 다수) + 대형 가전사 진입 가능성
- **진입 장벽 강화 전략**:
  1. **RFID Lock-in 생태계 조기 구축** — 정품 오일만 동작하는 폐쇄형 생태계로 전환 비용 극대화
  2. **AI 취향 데이터 누적** — 사용 기간이 길수록 레시피 품질 향상 → 이탈 시 데이터 손실로 Lock-in 심화
  3. **브랜드 IP·특허 선점** — PWM 드롭 제어 알고리즘, 다채널 혼합 조향 방식 특허 출원
  4. **유통 독점 파트너십** — 29CM·EQL 단독 입점, 와디즈 크라우드펀딩 최초 공개로 희소성 연출

## 5. Twelve Senses 차별화 포인트 강화 제안

### 5-1. "AI 조향 플랫폼"으로 카테고리 재정의

단순 스마트 디퓨저가 아닌 **"나만의 향을 만드는 AI 조향 플랫폼"** 으로 카테고리를 새로 정의한다. 경쟁사들은 기존에 정해진 향을 편리하게 쓰는 것에 집중하지만, Twelve Senses는 향 자체를 만드는 경험을 제공한다. 이 차이가 대체재 위협을 낮추고 프리미엄 가격 정당성을 확보하는 핵심 논거다.

- **실행 제안**: 온보딩 시 "향 취향 인터뷰" (5~7개 질문) → AI 첫 레시피 3종 제안 → 평가 반응으로 지속 학습
- **강화 포인트**: "이 레시피는 당신만의 것입니다" 메시지로 소유감 극대화

### 5-2. RFID Lock-in 생태계를 "정품 경험"의 프리미엄으로 포지셔닝

RFID를 단순 보안 수단이 아닌 **"정품 에센셜 오일의 품질 보증서"** 로 브랜딩한다. 소비자가 "락인이 불편하다"가 아닌 "정품이라 믿을 수 있다"로 인식하도록 커뮤니케이션을 설계한다.

- **실행 제안**: 앱에서 RFID 스캔 시 오일 원산지·성분·인증 정보 즉시 표시
- **강화 포인트**: 럭셔리 브랜드(에르메스·샤넬)의 정품 인증 경험 벤치마킹 — "태그 하나로 진품 확인"

### 5-3. B2B 무인 관리 SaaS — 국내 유일 포지션 선점

현재 국내 경쟁사 중 B2B 향기 관리 SaaS를 제공하는 플레이어가 없다. 호텔·백화점·리조트 대상 **원격 모니터링 + 자동 발주 + 공간별 레시피 A/B 테스트** 기능은 차별화된 수익원이자 진입 장벽이다.

- **실행 제안**: 2026 Q4 이후 B2B 파일럿 계약 2~3개소 확보 → 케이스 스터디 자산화
- **강화 포인트**: "향기 ROI 리포트" 제공 — 향기 변경 전후 매출·체류 시간 변화 데이터 제공

### 5-4. 날씨·컨텍스트 연동 자동 발향 — 경쟁사 미구현 영역

Pura는 스케줄 설정까지만, Xiaomi는 15가지 패턴 중 선택까지만 가능하다. Twelve Senses는 **날씨 API + 사용자 달력 + 시간대 + 기분 로그** 를 통합하여 컨텍스트 인식 자동 발향을 구현할 수 있다.

- **실행 제안**: 비 오는 날 → 우디/머스크 향 자동 전환, 월요일 아침 → 시트러스 집중 레시피 등 기본 룰셋 + AI 개인화 레이어
- **강화 포인트**: "오늘의 향 추천 알림" 푸시로 일일 앱 접촉률 확보
